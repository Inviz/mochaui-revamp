Introduction

Mocha is a web applications user interface library built on the Mootools javascript framework. The Mocha GUI components are made with canvas tag graphics.

Explore the navigation bar and see the windows to the right for more information.

- Greg Houston

Overview

Features

  • No images. The windows, including their controls, gradients and shadows, are drawn with the canvas tag.
  • Windows can be focused, dragged, resized, maximized, minimized, restored and closed.
  • Adjustable rounded corner radius.
  • Dynamically create new windows on demand.
  • Create modal dialog windows
  • Optional "desktop" toolbars
  • Minimal HTML markup required.
  • Tested in Firefox 2, Internet Explorer 6 & 7, Safari 2, and Opera 9.
  • Validates as XHTML 1.0 Strict if you use no target attributes in your anchors and validates as XHTML 1.0 Transitional if you do.
  • Gracefully degrades without javascript.

Bugs

  • Scrollbars show through GUI in Firefox on the Mac. This is a known bug.
  • In Opera 9, though the scrollbars in windows do not show through windows with a higher z-index, you can grab and scroll a scrollbar in a window that is behind another window with a higher z-index.
  • Also see the tickets on Google Code.
